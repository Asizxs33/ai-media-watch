# Spectra AI — Система умного мониторинга медиа-пространства РК

**Spectra AI** (Stitch AI Media Watch Monitor) — это специализированное мультимодальное IT-решение на базе искусственного интеллекта (MVP), разработанное для автоматического сбора на постоянной основе и анализа видео-, аудио- и текстового контента из социальных сетей (TikTok, Instagram, YouTube) с целью выявления признаков незаконного игорного бизнеса (онлайн-казино), финансовых пирамид, MLM-схем и различных видов интернет-мошенничества в Республике Казахстан.

Проект разработан в полном соответствии с требованиями и критериями хакатона по теме **«AI Media Watch: выявление признаков незаконного игорного бизнеса и финансовых пирамид в социальных сетях»**.

---

## 🚀 Основные возможности системы (MVP)

1. **Мультимодальный конвейер анализа контента**:
   * **Голос (Audio Speech-to-Text)**: Извлечение звуковой дорожки из видео (`yt-dlp`) и автоматическое транскрибирование речи через локальный Whisper.
   * **Визуальный ряд (Computer Vision / OCR)**: Анализ скриншотов постов, сторис или рекламных объявлений на предмет текста и графических вотермарок.
   * **Семантический анализ**: Выявление скрытых паттернов вовлечения (обещания легкого заработка, реферальные ссылки, сигналы трейдеров).
2. **Интерактивная карта угроз Казахстана**:
   * Разработана точная географическая карта КЗ (`KazakhstanMap.tsx`), отражающая концентрацию угроз и мошеннических кампаний по регионам в реальном времени.
3. **Реестр мошеннических реквизитов (Registry)**:
   * Автоматический сбор и группировка извлеченных платежных и контактных данных нарушителей:
     * Номера Kaspi/телефонов.
     * Номера банковских карт (16 цифр).
     * Telegram/WhatsApp контакты и ссылки.
     * Крипто-кошельки (USDT, BTC).
     * Сайты-зеркала казино и промокоды.
4. **Ассистент кибербезопасности & Голосовое управление**:
   * Встроенный AI-аналитик (`AnalystDock`), комментирующий инциденты и дающий советы оператору.
   * Управление командным центром голосом (через Web Speech API) или встроенную консоль CLI.
5. **Фоновый сбор данных (Chrome Extension)**:
   * Браузерное расширение сканирует медиа-активность пользователя, транскрибирует аудио потоком, классифицирует тексты в реальном времени и автоматически пополняет базу инцидентов.

---

## 🧠 Двухуровневая гибридная AI-модель (Критерий независимости)

Система построена по принципу **Edge-Cloud Hybrid** для обеспечения полной независимости и соответствия требованиям информационной безопасности:

1. **Автономный локальный уровень (100% Offline)**:
   * **Локальный классификатор текста**: Построен на базе библиотеки `scikit-learn` (TF-IDF векторизация + логистическая регрессия). Обучен на русскоязычном датасете мошеннического контента. Работает на сервере без внешних сетевых запросов.
   * **Локальная транскрипция речи**: Использует локальный порт модели Whisper (`faster-whisper`, модель `base`) на CPU/GPU, что гарантирует конфиденциальность обрабатываемых аудиоматериалов.
2. **Облачный уровень (AI Co-pilot)**:
   * Использует API **Claude** (модель `claude-haiku-4-5-20251001`) для глубокого лингвистического анализа двусмысленных фраз, выделения реквизитов из текста и оптического распознавания изображений (Claude Vision OCR).
   * **Отказоустойчивость**: При отсутствии интернет-соединения или API-ключа бэкенд бесшовно переключается на автономный локальный классификатор.

## 🛠 Технологический стек

* **Frontend**: React 19 + TypeScript + Vite + Tailwind CSS v4 + Framer Motion (интерфейсные анимации) + Recharts (аналитические графики).
* **Backend**: Node.js + Express (API-шлюз, координация ML-скриптов и базы данных).
* **Database**: PostgreSQL (Neon Server) для централизованного хранения инцидентов и черных списков реквизитов.
* **ML Service**: Python 3.10+ (классификация текстов `scikit-learn`/`joblib`, транскрипция `faster-whisper`, парсинг метаданных `yt-dlp`).
* **Chrome Extension**: Manifest V3 (Vanilla JS, фоновый скрипт сбора и аудио-перехвата).

## 📦 Инструкция по развертыванию и запуску

### 1. Подготовка окружения
Убедитесь, что на компьютере установлены:
* Node.js v18+
* Python 3.10+ (с установленными `pip`, `virtualenv`)
* Зависимости ОС для скачивания видео (утилита `yt-dlp` должна быть в PATH)

### 2. Запуск бэкенда (`backend`)
1. Перейдите в директорию `backend`:
   ```bash
   cd backend
   npm install
   ```
2. Создайте файл `.env` на основе примера `.env.example` и укажите параметры подключения:
   ```env
   PORT=3001
   DATABASE_URL=postgres://...  # Ссылка на PostgreSQL
   ANTHROPIC_API_KEY=sk-ant-...  # (Опционально) Ключ Claude API
   OPENAI_API_KEY=sk-...         # (Опционально) Ключ OpenAI Whisper
   ```
3. Запустите бэкенд в режиме разработки:
   ```bash
   npm run dev
   ```

### 3. Настройка и обучение локальной ML-модели (`ml`)
1. Перейдите в папку `ml`:
   ```bash
   cd ml
   pip install -r requirements.txt  # Установка sklearn, joblib, faster-whisper, numpy
   ```
2. Обучите локальную модель классификации текстов:
   ```bash
   python train.py
   ```
   *Скрипт сгенерирует обученную модель `model.pkl` и файл метаданных `model_meta.json`.*

### 4. Запуск фронтенда (`ai-media-watch`)
1. Перейдите в папку `ai-media-watch`:
   ```bash
   cd ai-media-watch
   npm install
   ```
2. Запустите dev-сервер:
   ```bash
   npm run dev
   ```
3. Откройте в браузере `http://localhost:5173`.

### 5. Установка Chrome-расширения (`chrome-extension`)
1. Откройте Google Chrome и перейдите на страницу `chrome://extensions/`.
2. Включите **«Режим разработчика»** (Developer mode) в правом верхнем углу.
3. Нажмите **«Загрузить распакованное расширение»** (Load unpacked) и выберите папку `chrome-extension` из корня проекта.
4. Настройте адрес бэкенда в попапе расширения (по умолчанию `http://localhost:3001`).

## ⚖ Законодательная база (Казахстан)

Классификатор Spectra AI настроен на детекцию нарушений согласно:
1. **Статья 217 УК РК** — Создание и руководство финансовой (инвестиционной) пирамидой.
2. **Статья 190 УК РК** — Мошенничество (в т.ч. интернет-мошенничество, фишинг, фейковые розыгрыши от Kaspi/Halyk).
3. **Закон РК «Об игорном бизнесе»** — Запрет деятельности нелицензированных онлайн-казино и букмекерских контор на территории страны.
